División

División de números decimales

DIVISIÓN DE UN NÚMERO DECIMAL ENTRE NÚMERO DECIMAL 

se realiza el siguiente procedimiento

  • se multiplican el dividendo y el divisor por la potencia de diez que tenga tantos ceros como el mayor número de cifras decimales que tenga el dividendo o el divisor.
  • como los números quedan sin cifras decimales, se realiza la división entre números naturales que queda sugerida. cuando se obtenga el residuo, se le agrega un cero y se pone la coma al cociente para seguir haciendo la división.

ejemplo:

para dividir 84,96 entre 2,4 se realiza.

la primera cifra tiene dos cifras decimales y la segunda cifra tiene una cifra decimal entonces se multiplica por 100.

84,96 x 100 = 8496 y 2,4 x 100 = 240

luego la división queda 8496/240 = 35,4

DIVISION DE UN NÚEMRO DECIAL ENTRE UNO NATURAL

 se realiza los siguientes pasos:

  • se realiza la división como si fueran números naturales.
  • se escribe una coma en el cociente cuando baja la primera cifra decimal del dividendo y se continua dividiendo de manera normal.

por ejemplo: 

para dividir 73,58/8=6,7225 

PARA DIVIDIR UN NÚMERO NATURAL ENTRE UNO DECIMAL

se deben realizar los siguientes pasos:

  • se multiplica el dividendo y el divisor por la potencia de diez  que tenga tantos ceros como cifras decimales tenga el divisor.
  • se realiza la división como si fueran números naturales.

ejemplo: 

para dividir 3278/ 1,5 se multiplican los dos números por 10.   

3278 x 10 = 32780    y 1,5 x 10 = 15, luego la división queda 32780715 =2191,3

Referencia bibliográfica:

Joya, A y otros. saberes ser hacer, matemáticas sexto. números decimales. pág. 114-133
